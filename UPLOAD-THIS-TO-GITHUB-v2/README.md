# Cardly NFC website

GitHub-klare statische website voor een dropshipping productpagina rond NFC-kaartjes.

## Productgegevens

- Product: zwarte blanco NFC kaartjes
- Pakket: 25 kaarten per set
- Prijs: €39,99 per set
- Checkout: Stripe-ready placeholder in `script.js`

## Online zetten

1. Upload deze volledige map naar GitHub.
2. Koppel de repository aan Vercel.
3. Gebruik als projecttype een gewone static site.
4. Vervang in `script.js` deze regel door je echte Stripe Payment Link:

```js
const stripePaymentLink = "https://buy.stripe.com/PLAATS-HIER-JE-STRIPE-LINK";
```
